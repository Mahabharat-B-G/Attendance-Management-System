#  Student Attendance Management System

A desktop-based application built with **Python**, **Tkinter**, and **MySQL** to efficiently track and manage student attendance. This system allows administrators to register students and log daily attendance with a clean, user-friendly interface.


##  Features
* **Student Registration:** Add new students with unique IDs, names, and courses.
* **Attendance Tracking:** Quick "Present/Absent" logging for registered students.
* **Database Integration:** Secure and persistent data storage using MySQL.
* **GUI Interface:** Simple, interactive window built with Tkinter (no command line required for daily use).


## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Database:** MySQL
* **GUI Framework:** Tkinter
* **Driver:** `mysql-connector-python`

##  Prerequisites
Before running the project, ensure you have:
1.  **Python** installed.
2.  **MySQL Server** installed and running.
3.  The **MySQL Connector** library. Install it via terminal:
    ```bash
    pip install mysql-connector-python
    ```
Configure Database Credentials
  host="localhost",
  user="your_username",
  password="your_password"
